# Включение SSH / root на роутерах Xiaomi и Redmi (MiWiFi)

[English](README.md) · **Русский**

Пошаговый гайд по включению SSH и получению root-доступа на роутерах Xiaomi / Redmi **без перепрошивки**. Проверено на **Redmi AX5400 Gaming Edition** со стоковой прошивкой **MiWiFi 1.0.95**.

> Сами шаги приведены только в [английской версии](README.md) — чтобы не дублировать их в двух местах. Раздел ниже поможет понять, подходит ли метод вашему роутеру.

## Поддерживаемые роутеры

Метод работает на нескольких моделях Xiaomi / Redmi с уязвимой стоковой версией MiWiFi (если у вас новее — сначала откатитесь). Рекомендуемая прошивка по моделям:

| Роутер | Версия MiWiFi |
| --- | --- |
| Redmi AX5400 Gaming/Esports Edition | 1.0.95 |
| Redmi AX3000 | 1.0.33 |
| Xiaomi AX3000 | 1.0.48 / 1.0.46 |
| Xiaomi AX1800 | 1.0.399 |
| Xiaomi AX9000 | 1.0.165 |
| Xiaomi AIoT AX3600 | 1.1.21 |
| Xiaomi AC2100 | 2.0.743 |
| Xiaomi AIoT AC2350 | 1.3.8 |
| Xiaomi 10 Gigabit Router | 1.0.53 |
